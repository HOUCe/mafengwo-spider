蚂蜂窝爬虫以及服务器

采用MEAN全栈架构

技术细节见  http://code.lujq.me/2015/09/12/%E8%9A%82%E8%9C%82%E7%AA%9D%E7%88%AC%E8%99%AB/

使用说明：

1. 安装Nodejs以及MongoDB
2. git clone git@github.com:golmic/mafengwo-spider.git
3. sudo npm install -g bower
4. npm install
5. bower install
6. node spider.js
7. node server.js
8. 用浏览器访问 http://127.0.0.1:4000
